Disable Microsoft Wi-Fi Sense

ABOUT
Modify the registry to disable Microsoft's Wi-Fi Sense

	
CONFIGURATION
The following options can be added to config.ini

	DISABLEMSWIFISENSE (Boolean) - Enable or disable the disable_MSWiFiSense plugin
		Y	- Enable the disable_MSWiFiSense Plugin (DEFAULT).
		N	- Disable the disable_MSWiFiSense Plugin.

NOTE
Disabling this plugin does not undo the changes made in previous runs.
